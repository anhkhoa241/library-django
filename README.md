# Hệ thống Quản lý Thư viện (Django)

### Cách cài đặt để chạy thử:
1. **Clone dự án:** `git clone <link-github-cua-anh>`
2. **Tạo môi trường ảo:** `python -m venv venv`
3. **Kích hoạt venv:** `source venv/bin/activate` (Linux) hoặc `venv\Scripts\activate` (Windows)
4. **Cài đặt thư viện:** `pip install django`
5. **Cập nhật database:** - `python manage.py makemigrations`
   - `python manage.py migrate`
6. **Tạo admin:** `python manage.py createsuperuser`
7. **Chạy server:** `python manage.py runserver`
